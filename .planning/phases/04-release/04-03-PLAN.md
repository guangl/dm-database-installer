cargo-dist 0.32.0 生成的 installer URL 模式（基于 Cargo.toml repository 字段）：
- `https://github.com/guangl/dm-database-installer/releases/latest/download/dm-database-installer-installer.sh`

（注：installer 脚本名以 [package].name 为前缀，本项目 name = "dm-database-installer"，二进制名 = "dm-installer"）

cargo-dist 生成的 curl|sh 推荐命令（来自 04-RESEARCH.md System Architecture Diagram）：
```
curl --proto '=https' --tlsv1.2 -LsSf \
  https://github.com/guangl/dm-database-installer/releases/latest/download/dm-database-installer-installer.sh | sh
```

PowerShell 等价命令（Windows）：
```
powershell -ExecutionPolicy ByPass -c "irm https://github.com/guangl/dm-database-installer/releases/latest/download/dm-database-installer-installer.ps1 | iex"
```

三个目标平台及对应的二进制 archive 名（cargo-dist 默认命名）：
- Linux x86_64: `dm-database-installer-x86_64-unknown-linux-gnu.tar.xz`
- Linux aarch64: `dm-database-installer-aarch64-unknown-linux-gnu.tar.xz`
- Windows x86_64: `dm-database-installer-x86_64-pc-windows-msvc.zip`

两个 install.sh 的区分（CONTEXT.md D-08）：
- `install.sh`（仓库根目录，Phase 1 产出，未完成）：DM8 数据库静默安装脚本
- `dm-database-installer-installer.sh`（cargo-dist 生成，发布到 GitHub Releases）：dm-installer 工具自身的下载安装引导

`run_with_runners` 的主流程按以下顺序：

1) 从 `specific` 中取 dminit、oguid、dsc_storage（`specific.dsc_storage.as_ref().ok_or_else(|| anyhow::anyhow!("缺少 dsc_storage"))?`）
2) 加载 checkpoint：`let mut cp = crate::cluster::checkpoint::ClusterCheckpoint::load()?.unwrap_or_default();`
3) Gate 1 — preflight：`if !cp.preflight_done { phases::run_preflight(&runners, dminit).await?; cp.preflight_done = true; cp.save()?; } else { tracing::info!("[续] 跳过预检查"); }`
4) Gate 2 — 安装：`if !cp.install_done { run_dsc_install_all_nodes(&common, &runners, dminit).await?; cp.install_done = true; cp.save()?; }`（见下方 helper）
5) Gate 3 — 分发 DSC 配置（并行所有节点）：`if !cp.dsc_config_distributed { run_distribute_dsc_configs_all_nodes(&runners, dminit, oguid, storage).await?; cp.dsc_config_distributed = true; cp.save()?; }`
6) Gate 4 — DMCSS + DMASM 启动（并行所有节点；DMCSS 先，DMASM 后）：`if !cp.css_asm_started { run_start_css_asm_all_nodes(&runners, dminit, &health_check_fn).await?; cp.css_asm_started = true; cp.save()?; }`
7) Gate 5 — ASM 初始化（仅 first_node）：`if !cp.asm_diskgroup_created { run_asm_init_first_node(&runners, dminit, storage).await?; cp.asm_diskgroup_created = true; cp.save()?; }`
8) Gate 6 — 共享 dminit（仅 first_node）：`if !cp.dminit_shared_done { run_dminit_shared_first_node(&runners, &specific.nodes, dminit, oguid, storage).await?; cp.dminit_shared_done = true; cp.save()?; }`
9) Gate 7 — config 目录分发（first_node → 每个 other_node）：`if !cp.config_dir_distributed { run_distribute_config_dirs(&runners, dminit).await?; cp.config_dir_distributed = true; cp.save()?; }`
10) Gate 8 — dmserver 注册启动并验证（所有节点先 first_node 后 others 串行，验证 V$INSTANCE）：`if !cp.dmserver_started { run_start_and_verify_dmserver_all_nodes(&runners, dminit, &health_check_fn).await?; cp.dmserver_started = true; cp.save()?; }`
11) `crate::cluster::checkpoint::ClusterCheckpoint::remove()?; tracing::info!("DSC 集群部署完成"); Ok(())`

私有 helper 函数（每个负责一个 Gate 的具体编排）：

- `async fn first_node_index(runners: &phases::Runners) -> Result<usize>`：返回第一个 role == Primary 的下标，否则 `bail!("DSC 集群缺少 primary 节点（first_node）")`
- `async fn run_dsc_install_all_nodes(common, runners, dminit)`：下载 PackageHandle（按 phases::run_install_phase 第 31-42 行模式），然后用 try_join_all 对每个节点调用 `deploy::run_dsc_install_only(&node, dminit, &pkg_path, runner.as_ref())`。注意：**不**调用 run_dminit_remote
- `async fn run_distribute_dsc_configs_all_nodes(runners, dminit, oguid, storage)`：try_join_all 遍历所有节点，每个节点调用 `deploy::distribute_dsc_configs(node, dminit, all_nodes, oguid, storage, node_index, runner.as_ref())`，其中 all_nodes 通过 `runners.iter().map(|(n, _)| n.clone()).collect::<Vec<_>>()` 得到，node_index 为该节点在 runners 中的下标
- `async fn run_start_css_asm_all_nodes(runners, dminit, health_check_fn)`：
  - 先在所有节点并行调用 `deploy::register_and_start_dmcss_service(&dminit.install_path, &format!("{}/dsc_conf/dmdcr.ini", dminit.install_path), runner.as_ref())`，等待全部完成
  - 再在所有节点并行调用 `deploy::register_and_start_dmasm_service(...)`，等待全部完成
  - DMASM 全部 start 后，对每个节点调用 `health_check_fn(node.host.clone(), 9349u16  /* DMASM 默认端口，DminitConfig 暂不扩展 */, 60).await?` 等待 DMASM 端口就绪（Pitfall 2 防御：dmasmtool 依赖 DMASM 共享内存就绪，仅 systemctl start 返回 0 不足够，需等待端口可达后才允许进入 Gate 5）
  - 严格顺序：DMCSS 完成 → DMASM start + 等待端口就绪 → 退出（Gate 5 的 dmasmtool/dmasmcmd 才可安全执行）
- `async fn run_asm_init_first_node(runners, dminit, storage)`：取 first_node runner，按顺序调用 `deploy::run_dmasmcmd_init(&dminit.install_path, storage, &format!("{}/dsc_conf", dminit.install_path), runner.as_ref())` 然后 `deploy::run_dmasmtool_create_diskgroups(&dminit.install_path, &format!("{}/dsc_conf/dmdcr.ini", dminit.install_path), runner.as_ref())`
- `async fn run_dminit_shared_first_node(runners, all_nodes, dminit, oguid, storage)`：取 first_node + runner，调用 `deploy::run_dminit_shared(first_node, all_nodes, dminit, oguid, storage, runner.as_ref())`
- `async fn run_distribute_config_dirs(runners, dminit)`：取 first_node 与 first_runner（Arc）。遍历其他节点（runners 中除 first_node 外的），对每个 other 调用 `deploy::distribute_config_dir(first_index, other_index, dminit, first_runner.as_ref(), other_runner.as_ref())`
- `async fn run_start_and_verify_dmserver_all_nodes(runners, dminit, health_check_fn)`：
  - 先在 first_node 调用 `deploy::register_and_start_dmserver_service(&dminit.install_path, &format!("{}/{}/dm.ini", dminit.data_path, first_node.instance_name), &format!("{}/dsc_conf/dmdcr.ini", dminit.install_path), first_runner.as_ref())`
  - 等待 first_node TCP 就绪：`health_check_fn(first_node.host.clone(), dminit.port, 60).await?;`
  - 再依次（不并行）启动其他节点 dmserver（port = dminit.port + node_index），等待 TCP 就绪
  - 最后并行对每个节点调用 `deploy::verify_dsc_node(&node, dminit, runner.as_ref())`

**关键约束**：所有 helper 函数遵循"读取 cp 字段→执行→cp.save()"的模式（rws/mod.rs::run_early_checkpoints 形态），但本次将 gate 直接展开在 `run_with_runners` 中，避免 RWS 模式中 `run_early_checkpoints / run_init_restore_checkpoints` 双层封装的复杂度。

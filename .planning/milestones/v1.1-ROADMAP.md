**Goal**: `dm-installer install rws` 端到端可走通，备库自动配置只读模式
**Depends on**: Phase 4 (v1.0)
**Requirements**: RWS-01, RWS-02
**Plans**: 3 plans

Plans:

- [x] 05-01-PLAN.md — ClusterCheckpoint 实现（新建 src/cluster/checkpoint.rs，5 个布尔字段 + serde_json roundtrip）
- [x] 05-02-PLAN.md — run_read_routing_phase + wait_for_standby_open（轮询 V$INSTANCE 直到 STATUS$=OPEN MODE$=STANDBY）
- [x] 05-03-PLAN.md — RWS checkpoint gate 集成：5 个 gate（preflight/install/primary_init/backup/standby_restore），完成后删除 checkpoint

**Details:**

Code review（第一次，Phase 05 完成后）修复 10 项：只读备库 SQL 配置缺失（CR-01）、SSH 端口硬编码（CR-02）、SYSDBA 明文密码（CR-03）、只读备库写操作（WR-01）、TOCTOU 竞态（WR-02）、verify 阶段顺序（WR-03）、最后一次轮询日志（WR-04）、println! vs tracing（IN-01/02/03）。

Code review（第二次，归档前）修复：install.sh 增加 check_port/check_memory 预检查、preflight.rs 磁盘要求 5→20 GB + 新增 check_memory、新增 env_setup.rs（dmdba 用户/SELinux/THP/timezone/limits）、standalone 安装前调用环境准备 + 安装后开启归档模式。

## Milestone Summary

**Pitfalls Defended (DSC):**

| Pitfall | Defense |
|---------|---------|
| P1: 不能调 run_install_phase（DMInstall.bin 不在共享存储上）| run_dsc_install_only 替代 |
| P2: DMASM 必须在 DMCSS 之后启动 | run_start_css_asm_all_nodes 串行顺序 |
| P3: SEQNO 按节点 index 不同 | generate_dmdcr_ini 动态生成 |
| P4: dminit 路径必须以 + 开头 | generate_dminit_ini 硬编码 + 前缀 |
| P5: V$INSTANCE 验证检查 OPEN 非 PRIMARY | verify_dsc_node 断言 OPEN |

**Key Decisions:**

| Decision | Rationale |
|----------|-----------|
| D-06: dmwatcher/dmmonitor 不适用于 DSC | DSC 共享存储节点直接访问共享数据，无需守护进程 |
| ClusterCheckpoint 无 install_path 匹配键 | rws.toml 和 checkpoint 同目录无混淆风险 |
| wait_for_standby_open_impl 参数化 | 避免 CI 120s 超时，保持生产常量完整 |
| DSC dsc_storage 段名与 TOML 键一致 | 与 [dsc_storage] TOML 段名保持一致 |

## v1.1 Requirements

### RWS 读写分离集群

- [x] **RWS-01**: 用户可执行 `dm-installer install rws` 完成读写分离集群完整部署（主库 + 只读备库，端到端）
- [x] **RWS-02**: 部署完成后备节点自动通过 SQL 配置为只读模式（READ_ONLY 标志）

### 状态查询

- [x] **STAT-01**: 用户可执行 `dm-installer status` 查询本地 DM 实例进程状态与端口监听
- [x] **STAT-02**: status 命令读取 config.toml 节点列表，通过 SSH 查询所有远程节点状态
- [x] **STAT-03**: 状态输出包含每个节点的进程状态、端口监听、数据库角色（PRIMARY/STANDBY/OPEN），格式为对齐表格

### DSC 共享存储集群

- [x] **DSC-01**: 用户可执行 `dm-installer install dsc` 完成 DSC 共享存储集群完整部署
- [x] **DSC-02**: 安装流程在所有节点自动调用 dmasmtool 初始化 ASM 磁盘组
- [x] **DSC-03**: 第一节点执行 dminit（路径指向共享存储），其他节点直接挂载启动
